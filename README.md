# 🦈 Pull Shark Practice Repository
This is my first pull request
This repository is created to practice and demonstrate GitHub workflows such as:.

* 🔀 Creating branches
* 📥 Opening Pull Requests
* ✅ Merging changes
* 🌱 Understanding version control

---

## 🚀 Purpose

The main goal of this repository is to:

* Improve GitHub collaboration skills
* Practice real-world development workflow
* Learn how pull requests work in a structured way

---

## 🛠️ Workflow Used

1. Created a new branch
2. Made changes to the README file
3. Opened a Pull Request
4. Reviewed and merged the PR

---

## 📌 Why This Matters

Pull Requests are a core part of modern software development.
They help in:

* Code review
* Collaboration
* Maintaining clean and stable codebases
